Full-Stack - Framework

Die Template-Library, das Layout-System und die Backendkomponenten sollen ein umfassendes Full-Stack-Framework werden.
Zudem soll das Front-End plattformunabhängig werden (d.h. man soll eine Webanwendung, aber auch eine Mobil-App, Desktop-Anwendung erzeugen können.
Desktop-Anwendungen sollen aber native UI-Elemente verwenden, d.h. es sollen für Buttons, Edits, Menüs usw. native Qt- oder Win32-Elemente anstatt HTML-Elemente genutzt werden.

Recursion-Library

Zeiterfassungs-Tool

Mit diesem Tool erfassen Mitarbeiter komfortabel ihre Arbeitszeiten.
Da jedes Unternehmen eigene Anforderungen – insbesondere in der Lohnabrechnung – hat,
ich die Software individuell an Ihre Bedürfnisse an.

Tender-Tool

Ein Tool zur Simulation von Zinstendern, wie sie im Banken- und Finanzbereich eingesetzt werden.
Durch Eingabe der gewünschten Zuteilung und der Gebote verschiedener Banken berechnet das System
automatisch den marginalen Zinssatz und zeigt an, wie die Mittel verteilt würden.
